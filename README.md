# Pokemon-Style Browser Game

A small Pokemon-style game with HTML Canvas and JavaScript.

I built this project to learn game devvelopment in a fun and practical way.

The battle start when you walk on the encounter zones .

---

## Features

- Walk around the town map
- Talk to NPC
- Enter Wild Battles
- Use attacks moves with effects
- Each and every attacks have different Damages
- view live HP bars int the battle filed
- Smooth music and sound effects

---

## Controls

| Key | Action |
|-----|--------|
| W / A / S / D | Move |
| Space | Talk / Continue dialogue |
| Mouse Click | Attack / Continue battle messages |

---

## Battle System

- Enter battles by walking through encounters zones
- Enemy appears and battle Ui transitions
- Player choose the monster they want
- Enemy attacks after player turn
- Battle ends when either side faints

---

## Move Types

- Damage moves : direct HP reduction
- Heal moves : Restores HP
- Shield moves : Absorbs incomming damages before HP reduction

---

## Character

- Charmander(player)
- Bulbasaur(player)
- Squirte(player)
- Draggle(Enemy)

---
## Tech Stack
- HTML
- CSS
- HTML Canvas
- JavaScript
- GSAP (animation)
- Howler.js(audio)
